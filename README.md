# Gestão de Aulas e Professores

Projeto de planilha para controle de aulas, disponibilidade de professores, restrições e impressão diária por professor.

## Funcionalidades
- Cadastro de professores
- Controle de disponibilidade por:
  - dia da semana
  - horário
  - modalidade (online, presencial ou ambas)
- Controle de restrições entre professor e aluno/turma
- Cadastro de aulas
- Verificação automática de:
  - disponibilidade
  - conflito de horário
  - restrições
- Status da escala
- Impressão das aulas do professor no dia

## Abas da planilha
- **PROFESSORES**
- **DISPONIBILIDADE**
- **RESTRICOES**
- **AULAS**
- **IMPRESSAO_PROFESSOR**

## Objetivo
Facilitar a organização de aulas flexíveis, ajudando na escolha de professores disponíveis e evitando conflitos de horário.

## Observações
A planilha foi pensada para uso no Excel e também adaptação para Google Sheets.
